# Multi-Frontend Command

Multi-model collaborative frontend development orchestrating Gemini (UI/UX authority) with Codex reference.

## Workflow Phases

1. **Research** - Requirement analysis
2. **Ideation** - Gemini UI/UX analysis
3. **Plan** - Gemini architectural planning
4. **Implementation** - Code development
5. **Optimization** - Gemini review
6. **Review** - Final quality checks

## Key Features

- **Multi-model collaboration**: Claude orchestrates, Gemini leads UI/UX
- **Session-based execution**: Resume capabilities across phases
- **Quality gates**: Score thresholds and user confirmation

## Usage

```
/multi-frontend <task-description>
```

## Communication Protocol

- Label responses with current mode
- Request user confirmation at phase transitions
- Background task execution with status monitoring
